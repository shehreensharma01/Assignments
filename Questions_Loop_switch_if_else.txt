2. Write a program that keeps asking the user to enter a positive number. Stop only when the user enters 0. Ensure the user is asked at least once (simulate do-while).
while True:
    num = int(input("Enter a positive number (0 to stop): "))
    if num == 0:
        print("Program stopped.")
        break
    else:
        print("You entered:", num)


3. Write a program using a while loop to calculate the factorial of a number entered by the user.
num = int(input("Enter a number: "))
factorial = 1
i = 1

while i <= num:
    factorial *= i
    i += 1

print("Factorial of", num, "is", factorial)

If score > 90: Grade A

If score > 60: Grade D

Else: Grade F

Use nested if-else.

if score > 90:
    grade = "A"
else:
    if score > 80:
        grade = "B"
    else:
        if score > 70:
            grade = "C"

5. Write a program that asks the user to enter a number from 1 to 7 and prints the day of the week using match-case:

1: Monday

2: Tuesday

...

7: Sunday

Any other number: Invalid input

match day:
    case 1:
        print("Monday")
    case 2:
        print("Tuesday")
    case 3:
        print("Wednesday")
    case 4:
        print("Thursday")
    case 5:
        print("Friday")
    case 6:
        print("Saturday")
    case 7:
        print("Sunday")
    case _:
        print("Invalid input")



6. Write a Python program that manages a list of student scores. Perform the following operations step-by-step:
Create an empty list to store scores.
Append the scores: 85, 90, 78, 92, 88
Insert the score 80 at index 
Remove the score 92 from the list
Sort the scores in ascending order
Reverse the list
Find and print the maximum and minimum score
Check if 90 is in the list
Print the total number of scores
Slice and print the first three scores
Iterate through the list and print each score

find the last element from the list
replace the score with new score on the index 2
create a new copied list also

# Create an empty list
scores = []

# Append scores
scores.append(85)
scores.append(90)
scores.append(78)
scores.append(92)
scores.append(88)
print("Scores after appending:", scores)

# Insert 80 at index 2
scores.insert(2, 80)
print("Scores after inserting 80 at index 2:", scores)

# Remove 92
scores.remove(92)
print("Scores after removing 92:", scores)

# Sort in ascending order
scores.sort()
print("Scores sorted in ascending order:", scores)

# Reverse the list
scores.reverse()
print("Scores after reversing:", scores)

# Find max and min
print("Maximum Score:", max(scores))
print("Minimum Score:", min(scores))

# Check if 90 is in the list
print("Is 90 in the list?", 90 in scores)

# Print total number of scores
print("Total number of scores:", len(scores))

# Slice first three scores
print("First three scores:", scores[:3])

# Iterate and print each score
print("All scores:")
for s in scores:
    print(s)

# Find last element
print("Last score in list:", scores[-1])

# Replace score at index 2
scores[2] = 95
print("Scores after replacing index 2 with 95:", scores)

# Create a new copied list
copied_scores = scores.copy()
print("Copied list:", copied_scores)
